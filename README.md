# Blog Website

A modern blog website built with Next.js. It features smooth animations, a clean UI, and a fully responsive design.

## 🚀 Features

- Animated UI using Framer Motion
- Responsive design (mobile + desktop)
- Email subscription system
- Clean and modern layout
- With dark and light theme
- Authentication using JSON Web Tokens (JWT)

## 🛠 Tech Stack

- Next.js
- React.js
- Tailwind CSS
- Framer Motion

## 🌐 Live Demo

https://blogger-sigma-dusky.vercel.app

## 📦 Installation

1. Clone the repository:

```bash
git clone https://github.com/tanveer-habib/Blogger.git
```

2. Navigate to the project folder:

```bash
cd blogs
```

3. Install dependencies:

```bash
npm install
```

4. Run the development server:

```bash
npm run dev
```

Open http://localhost:3000 in your browser.
